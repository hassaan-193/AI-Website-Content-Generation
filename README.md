# AI-Website-Content-Generation
AI-powered website content generator that interviews businesses and creates professional copy for Home, About, Services &amp; Contact pages. Built with Python/Flask + Groq AI. Reduces content creation time by 80%. Perfect for small businesses, freelancers &amp; startups who need quality web content fast.

AI Website Content Generator is a full-stack web application that leverages Large Language Models (LLMs) to automate the website copywriting process. Instead of spending hours writing or paying expensive copywriters, users answer a few smart questions and receive professional content for their entire website.
